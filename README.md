## Deskripsi Sistem

Sistem ini digunakan untuk mempermudah proses pengajuan kegiatan sosialisasi di kelurahan.
Admin dapat mengelola data pengajuan

## Fitur Sistem

- Login Admin
- Input Pengajuan Sosialisasi
- Edit & Hapus Data
- Lihat Data Pengajuan

## Teknologi yang Digunakan

- PHP
- MySQL
- HTML, CSS
- XAMPP

## Database

File database terdapat pada:

```
sosialisasi_kelurahan.sql
```

## Cara Menjalankan

- Import file `database.sql` ke phpMyAdmin
- Jalankan XAMPP (Apache & MySQL)
- Letakkan folder project di `htdocs`
- Buka di browser:

```
http://localhost/sosialisasi
```
